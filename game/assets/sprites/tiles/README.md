# Tiles

Tiles 2D usados pelo grid tatico da vertical slice.

- `ferraria-base-*`: variacoes de piso da Cidade-Forja de Ferraria.
- `ferraria-contamination-*`: variacoes visuais de contaminacao/Breach.
- `ferraria-obstacle-*`: obstaculos e cobertura.
- `floor.png` e `obstacle.png`: placeholders anteriores mantidos como fallback historico.
