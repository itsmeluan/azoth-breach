# 01 — Tratados Técnicos

Pasta destinada a tratados técnicos derivados dos documentos constitucionais.

Escopo previsto:

- física aplicada da alquimia;
- transmutação;
- energia;
- Azoth aplicado;
- materiais;
- biomodificação;
- homúnculos;
- artefatos;
- Pedra Filosofal;
- Pedra Filosofal Perfeita;
- Código Geométrico;
- protocolo universal da alquimia;
- pesquisa científica;
- implementação técnica de sistemas derivados do cânone.

Nenhum tratado técnico pode contradizer os documentos em `/docs/00_constituicao`.
