# 04 — Gameplay

Pasta destinada a documentação de game design e sistemas jogáveis.

Escopo previsto:

- core loop;
- progressão;
- combate;
- crafting;
- pesquisa;
- exploração;
- coop;
- PvP;
- economia;
- UI;
- UX.

Todo sistema de gameplay deve respeitar transmutação, conservação da massa-energia, Azoth, Código Geométrico, Protocolo Flamel, Especificação da Transmutação, Ruído Informacional, Véu, Breaches e demais conceitos constitucionais.

Após a Grande Transmutação Planetária, toda mecânica de transmutação deve assumir a arquitetura Consciência, Intenção, Protocolo Flamel, Especificação da Transmutação, Azoth, Transformação Física e Resultado.
