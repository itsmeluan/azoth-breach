# 98 — Consolidados GPT

Pasta auxiliar com documentos consolidados por camada documental para uso em ingestão, consulta agregada e fontes operacionais em ferramentas externas.

## Regra de Uso

Os arquivos desta pasta não substituem os documentos canônicos ativos.

Eles apenas agregam, em um único arquivo por camada, o conteúdo já existente nas pastas ativas de `/docs`.

Em caso de conflito, prevalecem sempre os documentos-fonte originais e a hierarquia oficial definida em `00_constituicao`.
